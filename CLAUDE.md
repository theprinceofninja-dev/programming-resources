# CLAUDE.md — Project Instructions

## Project Overview
This is a collection of programming learning resources originally saved as messages in a Telegram channel.
The goal is to sort, clean, and organize all messages into structured, readable files.

## Workflow
- Messages are provided in batches (e.g., "first 40 messages", "next 50 messages")
- For each batch: extract, clean, and sort content into the appropriate files
- After each batch, update INDEX.md to reflect new coverage and any new files

## File Structure

| File | Purpose |
|---|---|
| `INDEX.md` | Master index — tracks all files and batch coverage |
| `roadmap.md` | Learning path, steps, tools, and project ideas |
| `mindset.md` | Mindset, thinking approach, motivation, and habits |
| `resources.md` | Links, YouTube channels, courses, tools |
| `quotes.md` | Notable quotes and selected insightful comments |
| `curriculum.md` | Structured multi-week study plans and project assignments |
| `jobs.md` | Real job descriptions saved to understand what skills employers ask for |
| `CLAUDE.md` | This file — instructions for Claude |

## Sorting Rules
- Add content to existing files, don't create new files unless a genuinely new topic emerges
- If a new file is needed, add it to INDEX.md immediately
- YouTube comments: keep only those with a unique insight or memorable point — discard noise (likes, replies, timestamps)
- Arabic notes: translate and merge into the relevant section rather than keeping as a separate block
- Raw links without context: add to resources.md with a best-guess label based on surrounding messages
- Duplicate ideas: merge, don't repeat
- Keep formatting consistent with existing files (markdown headers, blockquotes for quotes, tables where appropriate)

## Language
- File content is in English
- Arabic notes are translated and integrated (not kept in Arabic)
